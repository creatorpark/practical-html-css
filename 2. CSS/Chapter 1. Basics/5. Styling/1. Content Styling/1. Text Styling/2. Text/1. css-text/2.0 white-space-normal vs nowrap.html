Normal vs Nowrap

New Lines

Original Text
Multiple     spaces collapse.
New lines collapse and text wraps to fit.
normal - collapse
Multiple spaces collapse. New lines collapse and text wraps to fit.
nowrap - collapse
Multiple spaces collapse. New lines collapse, no wrapping occurs.

Spaces and Tabs

Original Text
This is a    tabbed   and    spaced    text example.
normal - collapse
This is a tabbed and spaced text example.
nowrap - collapse
This is a tabbed and spaced text example.

Text Wrapping

Original Text
This text will wrap when it reaches the edge of the container.
normal - wrap
This text will wrap when it reaches the edge of the container.
nowrap - no wrap
This text will not wrap, even when it reaches the edge of the container.
