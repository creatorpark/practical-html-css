Text Align - Match Parent Example

Parent: text-align left
Child with match-parent alignment. The child text aligns to the left because the parent is left-aligned.
Parent: text-align right
Child with match-parent alignment. The child text aligns to the right because the parent is right-aligned.
Parent: text-align center
Child with match-parent alignment. The child text aligns to the center because the parent is center-aligned.
Parent: text-align justify
Child with match-parent alignment. The child text aligns justified because the parent is justified.
