Normal (Default Behavior)

This is normal text handling. Multiple spaces are collapsed into one, and the text will automatically wrap when it reaches the edge of the container. It's perfect for paragraphs of content.

Pre (Preserve Formatting)

Preserves formatting, spaces, and new lines exactly as typed. Useful for code blocks, poetry, or any text where layout is critical: function sampleCode() { console.log("This code block preserves indentation and spaces."); }

Pre-wrap (Text Editor Style)

This is a text editor simulation using pre-wrap. Notice how the line breaks are preserved: function sampleCode() { console.log("Preserves all spaces and line breaks while still wrapping text."); } Use this style when formatting matters, but you still need the text to fit within the constraints of your UI.
