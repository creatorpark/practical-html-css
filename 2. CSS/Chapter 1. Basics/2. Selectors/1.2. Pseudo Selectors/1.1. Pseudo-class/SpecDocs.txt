Pseudo-class는 Pseudo-elements와 다르게 특별한 스펙 문서가 없다.
그도 그럴 것이 기존에 있는 HTML에 정의된 Elements의 상태값을 기준으로
스타일링을 적용하는 것이기 때문일 것이다.
Elements를 스타일링 하는 것이기 때문에... 
그냥 원래 스타일링 룰을 따른다.


HTML Spec 문서
https://html.spec.whatwg.org/multipage/semantics-other.html#pseudo-classes

CSSOM Spec 문서
https://www.w3.org/TR/selectors-4/#pseudo-classes

HTML Spec 문서를 바탕으로 
자주 쓰는 Pseudo class 예제를 만든다.
