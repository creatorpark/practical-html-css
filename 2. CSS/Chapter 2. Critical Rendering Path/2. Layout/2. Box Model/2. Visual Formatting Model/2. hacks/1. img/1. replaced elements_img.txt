Replaced Element는 이미지, 아이콘, SVG와 같이 외부 콘텐츠를 웹 페이지에 
포함하는 특별한 요소 유형입니다. 
이러한 요소는 **고유 크기(Intrinsic Dimensions)**를 가지고 있으며, 
이는 콘텐츠 자체의 크기와 비율을 의미합니다.

1. Intrinsic Dimensions 개념:
- Replaced Element는 display 속성을 명시적으로 설정하지 않아도 고유 크기를 가지고 있습니다.
- 이는 이미지의 경우 실제 파일 크기를 의미하며, SVG의 경우 SVG 문서 내 크기를 의미합니다.
- Replaced Element의 고유 크기는 콘텐츠 자체의 크기와 비율을 정의하며, 레이아웃 계산 및 스타일 적용에 영향을 미칩니다.

-> 그래서 이를 관리하기 위해 명시적인 크기와 비율을 직접 적용 해야한다.
