Modular CSS with SASS (또는 SASS & Modular CSS)

설명: SASS를 통해 CSS를 모듈화하고, 변수와 믹스인을 활용한 재사용성 높은 코드를 작성하는 방법을 학습합니다.
구성:
1.1 Variables (변수)
1.2 Mixins (믹스인)
1.3 Nesting (중첩)
1.4 Partials & Imports (분할 및 가져오기)
1.5 Functions & Operators (함수 및 연산자)




BEM과 SASS 통합 학습:
학습 후반부에서 BEM과 SASS의 통합 활용 사례를 다루는 것도 좋습니다. 예를 들어, BEM 방법론을 SASS로 구현하는 방법을 다루면, 학습자들이 두 기술을 함께 사용하는 방법을 이해할 수 있습니다.
At Rules의 확장:
@supports를 이용한 조건부 스타일링이나 @layer와 같은 최신 CSS 기능도 함께 학습하면 좋을 것입니다.
