1. Box
HTML 요소가 렌더링할 때 차지하는 사각형 영역을 Box라고 한다.
CSS에서 스타일링 할 수 있는 모든 HTML 요소는 하나의 Box로 취급 된다.

2. Box Model
이 Box를 계산하는데 사용하는 개념적 모델을 Box Model 이라고 한다.

3. Box Model은 크게 2가지 Dimension과 Type으로 나눌 수 있다.

3-1. Dimension
- 요소의 크기를 정의
- content, padding, border, margin 등이 있습니다.

3-2. Type(display)
- 요소의 렌더링 되는 방식을 정의
- 크게 Block-level Boxes와 Inline-level Boxes로 렌더링 된다.

- HTML Elements를 Box화 할 때 크게 2개의 레벨로 다뤄진다.
3-2-1. Block-level Elements
- div, section, p 
- Block-level Boxes로 변환 된다.
- 기본적으로 블록 전체를 차지하며, 새로운 라인에서 시작합니다.
3-2-2. Inline-level Elements
- span, a, strong 
- Inline-level Boxes로 변환된다.
- 같은 라인에 다른 요소들과 함께 배치됩니다.
