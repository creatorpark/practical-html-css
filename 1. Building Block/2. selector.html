` 요소 내부의 모든 `

` 요소를 선택합니다. 즉, `

` 요소의 자손인 모든 `

` 요소에 스타일이 적용됩니다. ```html

This paragraph has background color #FFEFD5

This paragraph also has background color #FFEFD5 because it's a descendant of a div

``` 2. 자식 선택자 (`div > p`): 이 선택자는 `

` 요소에만 스타일이 적용됩니다. ```html

``` 3. 일반 동위 선택자 (`div ~ p`): 이 선택자는 `
` 요소 다음에 오는 모든 형제 `

``` 4. 인접 동위 선택자 (`div + p`): 이 선택자는 `

` 요소 바로 다음에 오는 `

``` 5. 속성이름 선택자 (`[title]`): 이 선택자는 `title` 속성을 가진 모든 요소를 선택합니다. 선택된 요소들에는 배경색이 검은색이고 글자색이 노란색인 스타일이 적용됩니다. ```html

This div has black background and yellow text because its title attribute is "first h2"
